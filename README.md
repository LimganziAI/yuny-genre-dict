# YUNY Genre Full-Body Dictionary

YUNY v2.7 음악 제작 시스템용 **장르 풀바디 사전** — 277개 장르를 카테고리별
개별 파일로 분할 (on-demand fetch 용). Suno AI 프롬프트 엔지니어링에 맞춰
확장·재구성한 자료.

구조: `23_GENRE_FULLBODY/<category>/<genre-slug>.md`
조회는 시스템의 슬림 인덱스(`23a_GENRE_INDEX_MASTER.md`)가 각 장르의 raw URL로 라우팅.
